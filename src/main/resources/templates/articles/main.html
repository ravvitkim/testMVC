url methode 출력내용 html page
/articles/list GET 전체게시글 읽기 성공 list_all.html
/articles/${id} GET [[${id}]]번 게시글 읽기 성공 list_one.html
/articles/create?name=tom&weight=80&height=180 GET [[${name}]]의 몸무게는 [[${weight}]] 이고 키는 [[${height}]]입니다. new.html
/articles/update GET 업데이트 성공 화면 출력 update_ok.html
/articles/${id}/update GET [[${id}]]번 게시글 수정 폼 로딩 완료 update.html
/articles/{id}/delete GET [[${id}]]번 글 삭제 완료 delete_ok.html
/articles/{id}/articleComment GET [[${id}]]번 게시글의 모든 댓글보기 성공 comment_view.html
/articles/{id}/articleComments/{article-comment-id}/delete GET [[${id}]]번 게시글의 [[${commentId}]]번째 답글 삭제 delete_ok.html
